# Celestials

Celestials is a modern, enterprise-focused Discord bot platform built with .NET.

## Goals

- Clean architecture and modular features
- Strong reliability and observability defaults
- Scalable request/rate-limit handling

## Getting Started

1. Install the .NET SDK used by this repository.
2. Restore dependencies.
3. Build the solution.

## Repository Structure

- `src/` — application and library projects
- `tests/` — test projects
- `docs/` — architecture, ADRs, and runbooks
